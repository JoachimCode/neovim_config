NvChad / Neovim Cheat Sheet

Leader key: Space. Daily coding essentials: selecting, copying/pasting, deleting, search, LSP.

Tip: open a code file to see LSP diagnostics Neovim 0.11+: :checkhealth vim.lsp

Selection visual mode

ActionKeysNotes
Select charactersvThen move cursor
Select full linesVLinewise visual
Block/column selectCtrl+vRectangular selection
Select word (fast)viwviw = “visual inner word”
Select N wordsv3wExample: v3w
Select while jumping wordsv then w/b/eSelection = movement

Big-word movement (treats snake_case as one word): W/B/E (uppercase).

Copy & Paste yank / put

ActionKeysNotes

Recommended: set vim.opt.clipboard = "unnamedplus" so y/p use the macOS clipboard.

Delete usually yanks too (so you can paste with p). Use "_d to not overwrite your last copy.

Terminal inside nvim

ActionKeysNotes
Exit terminal mode (recommended)EscMap it: vim.keymap.set("t","<Esc>", [[<C-\><C-n>]])
Fallback<C-\><C-n>Hard on some mac layouts
Close terminal window:qAfter normal mode

If Esc doesn’t exit: add the mapping in lua/mappings.lua and restart.

Best accuracy: your real keybinds are in ~/dotfiles/nvim/lua/mappings.lua.
